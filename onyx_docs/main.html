Setup

To install the Onyx library, run pip install onyx in the terminal. Make a new file wherever and call it whatever you want, just make sure it has a .py extension.
Inside that file, add the following lines:

from onyx import * Handler(datapack_path)


Functions

Minecraft functions are created very similar to Python. The below segment will create a function in D:/onyx_docs/data/onyx_docs/functions/foo/bar.mcfunction

from onyx import * Handler("D:\onyx_docs") @path("foo") def bar(): say("Hello, World!")

You can also add a function to tick.json (so it runs twenty times per second) by using the @loop decorator like so:

@loop @path("foo") def bar(): say("Hello, World!")

Be advised that the loop decorator must be put above the path decorator.
